先来认识一下内联元素和块级元素

inline          内联元素，不能设置  width  height  margin-top/-bottom  padding-top/-bottom

block          块级元素   可以设置折现属性，但是他独占一行，并且宽度自动填满父元素的宽

inline-block  同时具备内联元素和块级元素的特性。

-----------------------------------------------------------------------------------------------------------

html标签间，出现一点缝隙，有可能会影响到我们想要的结果。
比如：  滚动图片。 有时候出现一丝的缝隙。
开始的时候我I试着样做的。

<ul id="scroll">
    <li style="display:inline-block"><img src="http://yhtqduan.blog.163.com/"/> <li>
       .........
</ul>

这是时候，我的li标签间，出现了大约3像素的缝隙，结果怎么都调不好，

以为是  border  padding  margin 在做怪，

后来才发现

是内联元素在做怪。

元素分为内联元素和块级元素

最后设置css

   li{  display:block; float:left; }   

完美解决

abstract 修饰的类是抽象类，
abstract 修饰的方法是抽象方法

abstract A {
      abstract  function get();     // 抽象方法不能有方法体，不是抽象方法可以有
      function set(){
        echo "这个抽象类的方法可以使用";
    }
}
$a = new A();    //  报错
1 ，  抽象类不能被 new 只能被继承

2，   抽象方法要在子类中重写，（重写：参数的数量不能变，类型必须对）

3，    如果有一个抽象方法，必须声明为抽象类

4，  抽象类可以继承抽象类，目的是为了扩展抽象类

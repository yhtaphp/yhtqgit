属性： 用来描述对象的数据元素（也称为 数据 / 状态）
方法 ：对对象的属性进行的操作  （也称  行为/ 操作）
private ： 属性 / 方法          # 只能目前的这个对象里可用。
ptotected : 属性 / 方法          #可以在目前对象  以及  子对象 中使用。
public         属性 / 方法          #可以在任何位置使用

对象内部使用：
$this -> xxx;

对象外部使用，需要实例话对象
$obj = new Obj();
$obj -> xxxx;

静态方法:
  外部使用 ： Obj :: xxx();
  内部使用 :    self :: xxx()

静态属性：
外部：Obj :: xxx;
内部： $this -> xxx;

简单实：

@media screen and (max-width:1000px){
	.test{
		width:1000px;
		height:500px;
		text-align:center;
		border:1px solid #0f0;
	}
}

当屏幕 小于1000xp（像素）时，使用里面的样式。


@media screen and (min-width:1000px){
	.test{
		width:200px;
		height:300px;
		color:#00f;
		border:1px solid #00f;
		text-align:center;
	}
}


屏幕大于1000px时，使用里面的样式。
